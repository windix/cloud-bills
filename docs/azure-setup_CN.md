# Azure 配置指南

要从 Azure 获取费用数据，您需要创建一个服务主体（应用注册），并在订阅级别授予其 **成本管理读者** 角色。

## 1. 创建服务主体

1. 登录 [Azure 门户](https://portal.azure.com/)。
2. 搜索 **应用注册** 并选择它。
3. 点击 **+ 新建注册**。
4. 输入名称（例如 `cloud-bills`）。
5. 将 **受支持的帐户类型** 设置为"仅限此组织目录中的帐户（默认目录）"。
6. 点击 **注册**。
7. 复制 **应用程序（客户端）ID** 和 **目录（租户）ID**。

## 2. 创建客户端密钥

1. 在应用注册中，进入 **管理 > 证书和密码**。
2. 点击 **+ 新建客户端密码**。
3. 添加描述并设置过期日期。
4. 点击 **添加**。
5. 立即复制客户端密码的 **值**（之后将无法再次查看）。

## 3. 分配权限（最小权限原则）

若要查询费用数据，服务主体需要 **成本管理读者** 角色。

1. 在 Azure 门户中搜索 **订阅**。
2. 选择您要监控的订阅。
3. 在左侧边栏点击 **访问控制 (IAM)**。
4. 点击 **+ 添加** → **添加角色分配**。
5. 搜索 **成本管理读者**，选中后点击 **下一步**。
6. 点击 **+ 选择成员**。
7. 搜索您的服务主体名称（例如 `cloud-bills`）。
8. 选中它，点击 **选择**，然后点击 **查看 + 分配**。

## 4. 配置文件

在项目根目录根据 `azure.yaml.example` 创建 `azure.yaml` 文件：

```yaml
default: main
main:
  tenant_id: "your-tenant-id"
  client_id: "your-client-id"
  client_secret: "your-client-secret"
  subscription_id: "your-subscription-id"
```
